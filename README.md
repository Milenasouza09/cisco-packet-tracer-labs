# 🌐 Cisco Packet Tracer Labs

Este repositório reúne os laboratórios e exercícios desenvolvidos durante meus estudos de Redes de Computadores utilizando o Cisco Packet Tracer.

Os projetos foram criados com o objetivo de praticar conceitos de redes, configuração de dispositivos e comunicação entre computadores.

---

## 📚 Conteúdo

- Configuração de redes locais (LAN)
- Endereçamento IP
- Configuração de switches e roteadores
- Testes de conectividade
- Simulações utilizando Cisco Packet Tracer

---

## 🛠️ Ferramentas

- Cisco Packet Tracer

---

## 🎯 Objetivo

Praticar conceitos fundamentais de Redes de Computadores e desenvolver experiência com ambientes simulados utilizando o Cisco Packet Tracer.
